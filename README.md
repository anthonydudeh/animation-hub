# Roblox Animation Hub

A secure, feature-rich animation hub system for Roblox that allows players to safely equip and use animation packs from the marketplace.

## Features

**Secure Animation Loading** - Server-side validation and whitelist system
**Modern UI** - Clean squircle design with smooth animations
**Anti-Cheat Protection** - Rate limiting, validation, and exploit detection
**Easy Integration** - Simple setup with minimal configuration
**Performance Optimized** - Efficient animation management

## Quick Start

1. Clone this repository
2. Copy the scripts to your Roblox game:
   - Server scripts go to `ServerScriptService/AnimationHub/`
   - Client scripts go to `StarterPlayer/StarterPlayerScripts/AnimationHub/`
3. Configure whitelisted animation pack IDs in `AnimationHubService.lua`
4. Test in your game!
